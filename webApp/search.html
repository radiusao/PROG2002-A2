How to use Home Page:

  • You can navigate to the search page using the navbar above
  • By default the home page only shows active fundraiser, to show both non-active and active fundraiser toggle the "show non-active fundraiser" blue button
  • To go to the fundraiser page, simply click the blue "view" button

How to use Search Page:

  • You can navigate back to home page using the navbar above
  • Use the "Show only Active" blue button toggle to adjust if you'd like to show active or both active and non-active fundraiser
  • Use the "Fundraiser Name blue button toggle to switch between searching by the fundraiser name or the caption of the fundraiser"
  • Typing on the text box will automatically search the data base according to your filter and search key
  • Changing the 2 blue filter button will also update the search
  • Use the red "clear" button to clear up the search page
